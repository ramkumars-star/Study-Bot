Syllabus StudyBot is a simple, self-contained chatbot designed to help students learn school or university syllabus topics in an easy and interactive way.

This application runs entirely in a single chatbot.html file and uses the Gemini API with Google Search grounding to provide factual, sourced, and easy-to-understand explanations.

Features

Topic-Based Learning: Start by defining a specific topic you want to study (e.g., "Photosynthesis," "Pythagorean Theorem").

Simple Explanations: Get complex subjects broken down into simple, step-by-step explanations.

Key Fact Generation: Instantly get a list of the most important facts and core concepts for your topic.

Interactive Quizzes: Test your knowledge with simple, on-demand multiple-choice questions.

Factual & Sourced: All answers are grounded using Google Search to ensure they are factually accurate. The bot provides source links for further reading.

Follow-up Questions: Ask specific follow-up questions in a natural, conversational way.

Dark/Light Mode: Includes a theme toggle to switch between dark and light modes, which is saved to your browser's local storage.

Single File, No Dependencies: Runs entirely from a single HTML file. There is no need for a backend server or complex setup.

How to Use

Open the File: Simply open the chatbot.html file in any modern web browser.

Enter a Topic: In the input box at the bottom, type the subject or topic you want to learn about and press "Start."

Choose an Action: Once the topic is set, you can use the action buttons:

Explain Simply: Get a simple breakdown of the topic.

Key Facts: Receive a numbered list of the most important facts.

Quiz Me: Get a multiple-choice question to test your understanding.

New Topic: Clear the current topic and start fresh with a new one.

Ask Questions: Use the chat input box (which appears after starting a topic) to ask any specific follow-up questions (e.g., "Why is that important?" or "What's an example of that?").

Technology Used

HTML5: For the main structure of the application.

Tailwind CSS: For all styling, providing a modern and responsive UI.

JavaScript (ES6+): To handle all application logic, UI interactions, and API calls.

Gemini API (gemini-2.5-flash-preview-09-2025): The generative AI model that powers the chatbot's responses.

Google Search Grounding: A feature of the Gemini API used to ensure all educational answers are based on factual, verifiable web sources.
